## Basic Syntax

These are the elements outlined in John Gruber's original design document. All Markdown applications support these elements.

### Headings

```markdown
# H1
## H2
### H3
```

### Bold

```markdown
**bold text**
```

**bold text**

### Italic

```markdown
*italicized text*
```

*italicized text*

### Blockquote

```markdown
> blockquote
```

> blockquote

### Ordered List

```markdown
1. First item
2. Second item
3. Third item
```

1. First item
2. Second item
3. Third item

### Unordered List

```markdown
- First item
- Second item
- Third item
```

- First item
- Second item
- Third item

### Code

```markdown
`code`
```

`code`

### Horizontal Rule

```markdown
---
```

---

### Link

```markdown
[Markdown Guide](https://www.markdownguide.org)
```

[Markdown Guide](https://www/markdownguide.org)

### Image

```markdown
![alt text](https://www.markdownguide.org/assets/images/tux.png)
```
